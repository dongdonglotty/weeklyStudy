### git config  相关

  就这个命令来说，通常的教程上都是教你这样配置的：  

```bash
# --global 表示当前用户级别的配置文件，在linux下是 ~/.gitconfig 或者 ~/.config/git/config
$ git config --global  


# --system 表示系统级别的配置文件，在linux比如说是 /etc/gitconfig 等等
$ git config --system


# --local 表示仓库级别的配置文件
$ git config --local
```
